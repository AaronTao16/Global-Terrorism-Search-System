# Create MERN application 

## Environment set up
``` bash
# install express
npm i express

# install monitor for development
npm i nodemon cors --save-dev

# set environment
npm i --save-dev dotenv

# install mongodb tool
npm i mongoose

```

## Quick start
```bash
# Install dependencies
npm install

# Run develop mode
npm run server

# Server runs on http://localhost:5000
```
